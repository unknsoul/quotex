Gap 12 — Regime Detection: EMA Stack Not Added ⚠️

The predict_engine.py now uses a basic 2-EMA gate, but regime_detection.py's detect_regime() function was not upgraded. It still classifies regime using only ATR + ADX + EMA-50 slope — missing the full EMA stack alignment check.
